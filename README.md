# Scalable Web Scraping System

## Overview
This project is a Python-based web scraping system designed to extract structured data from multi-page websites.

## Key Features
- Multi-page scraping with pagination handling
- Data cleaning (encoding fixes and formatting)
- Structured dataset generation (CSV output)
- Error handling for reliable execution

## Technologies Used
- Python
- BeautifulSoup
- Requests

## Workflow
1. Fetch webpage data
2. Parse HTML content
3. Extract structured information
4. Clean and validate data
5. Store results in CSV format

## Output
Extracted fields:
- Title
- Price
- Availability

## Use Case
This project simulates real-world data extraction workflows used in AI systems, data engineering pipelines, and automation tasks.
